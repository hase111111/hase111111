研究

私が大学で専攻している分野は「ロボット工学」です．中でも運動軌道生成，軌道追従制御，機構解析などを研究しています． 担当教師は琴坂信哉（KOTOSAKA, Shinya）准教授です．
移動形態に脚歩行を採用した多脚ロボットの歩行や，冗長自由度を持つロボットハンドの制御など， 複雑な運動学・動力学を必要とするロボットの制御に関する研究を行っています．

グラフ探索による自由歩容生成

大学4年時に行った研究です． 6つの脚を持つ多脚ロボットの自由歩容生成を目的として，グラフ探索を用いた手法を提案しました．

キーワード：多脚ロボット，自由歩容生成，グラフ探索
協働ロボットのための軌道生成

現在取り組んでいる研究です． 人間と協働するロボットのための軌道生成手法を研究しています．

キーワード：協働ロボット，ロボット安全，冗長双腕ロボット

ゲームプログラミング

幼いころから「オリジナルのゲーム作りをしたい」という夢があり，画用紙にオリジナルのゲームを描いたり，自由帳を架空の自作ゲームの攻略本にしたりして遊んでいました． PCを手に入れた中学生の頃からプログラミングを始め，中学3年生の頃にはじめてC++とDxLibを使ってゲームを作りました． 以降，今に至るまで，C++，C#，Unity，Unreal Engineなどを使ってゲームを作り続けています． 個人的に趣味で作成したゲームや，大学生になってから，所属しているサークルで作成したゲームを以下に紹介します．

ばーど だっしゅ!

Unreal Engine5で作成した3Dアクションゲームです． 1人で，8か月程度かけて開発しました． 個人開発したゲームでは珍しく，ゲーム中に登場するほぼすべての3Dモデルを自作しているのが特徴です． それにともない，キャラクターのリグ，アニメーションも自作しています．

2022年(大学3年時)に作成．文化祭で発表．
Carry Block !

Unityで開発した3Dパズルゲームです． 10人程度のサークルのメンバーと共に，約1年かけて開発しました． 私はプランナー，3Dモデラー，アニメーターとして，多岐にわたって開発に携わりました． 文化祭の他，GAME^3（ゲームキュービック），ゲームプログラマー甲子園などさまざまなイベントで発表しました．

2024年(大学4年時)に作成．文化祭や各種イベントで発表．
マッチ棒ゲーム

DxLibを使って作成したデジタルボードゲームです． 1人で，5か月程度かけて開発しました． 有名な手遊びをデジタル化したもので，CPUと対戦できます． min-max法を用いたCPUの思考ルーチンを実装しており，最高難易度に人間が勝つことは不可能です． 文化祭に向けて開発しましたが，新型コロナウイルス感染症の影響で文化祭が中止となったため，2023年度以降の文化祭で発表しています．

2022年(大学2年時)に作成．
めりんちゃんのホームランダービー

大学2年次まで，新型コロナウイルス感染症の影響で対面のサークル活動が制限されていたため，大学3年次の新入生歓迎会のために作成したゲームです． DxLibを用いて作成し，1人で4か月程度かけて開発しました． 埼玉大学のマスコットキャラクター「めりんちゃん」を主人公にし，有名な野球ゲームをモチーフにした2Dゲームです．

2022年(大学2年時)に作成．
その他のゲーム
